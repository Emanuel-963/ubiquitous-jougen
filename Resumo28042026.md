## O que é o IonFlow Pipeline?

O **IonFlow Pipeline** é uma plataforma open-source de análise eletroquímica desenvolvida em Python, destinada a investigadores em baterias, supercapacitores e ciência de materiais. O software integra num único ambiente:

- Espectroscopia de Impedância Eletroquímica (EIS)
- Distribuição de Tempos de Relaxação (DRT)
- Ciclagem Galvanostática
- Análise comparativa multi-amostra com Health Score e PCA
- Relatórios automáticos com interpretação por IA (GPT / Ollama)
- Base de dados SQLite local para rastreabilidade completa
- Dashboard web Streamlit para visualização interativa

## Funcionalidades por Módulo

### Parsers (`src/parsers/`)
- Auto-deteção de formato por extensão + magic bytes
- Retorna `ParsedEIS` com `.data` (DataFrame) e `.metadata` (dict)

### Circuit Fitting (`src/circuit_fitting.py`)
- 10+ circuitos pré-definidos: Randles, Two-Arc, CPE, Indutivo...
- Auto-composição de circuitos combinatórios
- Scoring multi-critério: BIC, confiança ML, validação Kramers-Kronig
- Ranking automático categorizado em português

### DRT (`src/drt_analysis.py`)
- Regularização de Tikhonov no domínio do tempo de relaxação
- Extração automática de picos τ e amplitude γ
- Visualização stem-plot + distribuição suavizada

### Ciclagem (`src/cycling_calculator.py`)
- Energia (Wh/kg), Potência (W/kg), Eficiência Coulômbica
- Retenção de capacidade (critério 80%)
- Diagrama de Ragone interativo

### Comparação (`src/comparison/`)
- Sobreposição de N amostras em Nyquist, Bode e DRT
- Health Score ponderado + PCA com clustering k-means
- Relatório comparativo PDF

### IA / LLM (`src/ai/`)
- `LLMAdapter` abstrato — OpenAI + Ollama + NullAdapter (offline)
- Narrativa científica gerada automaticamente
- Dashboard: página de IA interativa no browser

### Base de Dados (`src/db/`)
- 7 tabelas SQLite com FK CASCADE e índices otimizados
- Migrações de schema versionadas e idempotentes
- `IonFlowRepository` e `FeatureStoreV2` com API limpa

### Dashboard (`dashboard.py`)
- 7 páginas Streamlit: Visão Geral, Upload, EIS, DRT, Ciclagem, Histórico, IA
- Acesso local ou em rede para apresentações laboratoriais
- Upload de ficheiros com pré-visualização Nyquist em tempo real

## Como Instalar e Usar

### Instalação (uma única vez)
```bash
git clone https://github.com/Emanuel-963/ubiquitous-jougen.git
cd eis_analytics
python -m venv venv

# Windows PowerShell:
.\venv\Scripts\Activate.ps1
pip install -e .[dashboard]

# Linux / macOS:
source venv/bin/activate
pip install -e .[dashboard]
```

### Lançar a Interface Gráfica (GUI)
```bash
python gui_app.py
```

### Lançar o Dashboard Web
```bash
.\venv\Scripts\python.exe -m streamlit run dashboard.py
# → Abre em http://localhost:8501
```

### Linha de Comandos (CLI)
```bash
ionflow-cli eis     --data-dir data/raw/ --output outputs/
ionflow-cli cycling --data-dir data/raw/ --output outputs/
ionflow-cli drt     --data-dir data/raw/ --output outputs/
ionflow-cli analyze --all --ai --export-pdf relatorio.pdf
```

### Usar via Python
```python
from src.parsers import parse_eis_file
from src.db import IonFlowRepository

df = parse_eis_file("medicao.dta").data

repo = IonFlowRepository("data/ionflow.db")
sid  = repo.add_sample("AM1", "eis")
repo.save_eis_results(sid, df_params)
```

## Fluxo de Trabalho Típico

1. **Colocar ficheiros EIS** em `data/raw/` (`.dta`, `.mpr`, `.mpt`, `.csv`, `.txt`, `.xlsx`)
2. **Executar o pipeline** na GUI ou CLI — resultados em `outputs/` e na BD SQLite
3. **Abrir o Dashboard** para visualizar e explorar (`localhost:8501`)
4. **Comparar amostras** na aba Comparar — Health Score + PCA + ranking
5. **Pedir interpretação** ao agente IA (OpenAI ou Ollama local)
6. **Exportar** tabela LaTeX + gráficos PNG 300 DPI para o artigo
