Este tutorial ensina como combinar os resultados dos três pipelines
(EIS, Ciclagem, DRT) para tirar conclusões científicas significativas.

--------------------------------------------------------------------------------
1. FLUXO DE ANÁLISE CIENTÍFICA COMPLETA
--------------------------------------------------------------------------------

  PASSO 1: Obter dados
  │
  ├── EIS: Importar para data/raw/ → Rodar Pipeline EIS
  ├── Ciclagem: Importar para data/processed/ → Rodar Pipeline Ciclagem
  └── DRT: Mesmos dados EIS → Rodar Pipeline DRT

  PASSO 2: Análise individual
  │
  ├── EIS: Rs, Rp, C, ranking, PCA, circuitos
  ├── Ciclagem: Energia, potência, retenção cíclica
  └── DRT: Processos de relaxação, picos, constantes de tempo

  PASSO 3: Análise cruzada (ESTE TUTORIAL)
  │
  ├── EIS ↔ Ciclagem: impedância prediz performance?
  ├── EIS ↔ DRT: circuito equivalente é consistente com DRT?
  └── DRT ↔ Ciclagem: quais processos limitam a estabilidade?

  PASSO 4: Conclusões para publicação

--------------------------------------------------------------------------------
2. CORRELAÇÕES EIS ↔ CICLAGEM
--------------------------------------------------------------------------------

2.1 Rs (Resistência série) ↔ Potência:

  EXPECTATIVA: Rs baixo → Potência alta
  POR QUÊ: Rs é a resistência ôhmica total (eletrólito + contatos + bulk).
  Quanto menor, menos energia é perdida como calor, permitindo
  maior taxa de carga/descarga.

  SE NÃO CORRELACIONAR:
  • Possível contribuição de difusão (não capturada pelo Rs)
  • Rs pode ser dominado por contato elétrico, não pelo material
  • Scan rate muito diferente entre EIS e ciclagem

2.2 Rp (Resistência de polarização) ↔ Energia:

  CORRELAÇÃO COMPLEXA:
  • Para supercapacitores: Rp alto pode significar mais
    armazenamento faradaico → MAIOR energia
  • Para baterias/proteção: Rp alto = maior barreira → MENOR taxa
  • Depende do mecanismo: EDLC vs pseudocapacitância vs faradaico

  COMO DISTINGUIR:
  → Se n (expoente CPE) > 0.9: comportamento capacitivo
  → Se n ≈ 0.5-0.7: pseudocapacitivo ou difusivo
  → Se DRT mostra pico em τ > 10 ms: processo difusivo dominante

2.3 Capacitância (C_mean) ↔ Energia específica:

  EXPECTATIVA: C_mean alto ↔ Energia (Wh/kg) alta
  POR QUÊ: E = ½CV² — energia armazena-se na capacitância.

  SE NÃO CORRELACIONAR:
  • C_mean do EIS é calculada numa faixa de frequência específica
  • A capacitância de ciclagem é "quasi-static" (DC)
  • A diferença pode indicar processos lentos não capturados pelo EIS

  DICA: Compare C_lowfreq (menor frequência do EIS) com Energia —
  é a melhor aproximação da capacitância DC pelo EIS.

2.4 Tau (Constante de tempo) ↔ Rate capability:

  EXPECTATIVA: τ baixo → melhor performance em alta taxa
  POR QUÊ: τ = RC — tempo necessário para carga/descarga.
  Materiais com τ curto podem ciclar mais rápido.

  COMO VERIFICAR:
  • Compare τ com a duração do ciclo de ciclagem
  • Se τ << duração do ciclo → EIS prediz bem
  • Se τ ≈ duração do ciclo → material no limite de sua cinética
  • Se τ >> duração do ciclo → material não consegue responder

2.5 Score/Rank ↔ Retenção:

  Gráfico "Rank vs Retenção" mostra diretamente esta relação.

  CENÁRIOS:
  • Correlação forte → Impedância prediz estabilidade → Publicar!
  • Sem correlação → Degradação vem de fatores não-impedimétricos:
    ─ Dissolução química do material ativo
    ─ Delaminação mecânica
    ─ Reações parasitas (decom. do eletrólito)
    → Requer técnicas complementares (SEM, XRD, etc.)

--------------------------------------------------------------------------------
3. CORRELAÇÕES EIS ↔ DRT
--------------------------------------------------------------------------------

3.1 Número de semicírculos (Nyquist) ↔ Número de picos DRT:

  EXPECTATIVA: Consistência
  • 1 semicírculo → 1 pico DRT principal
  • 2 semicírculos → 2 picos DRT
  • Semicírculos sobrepostos → DRT revela picos que o Nyquist esconde!

  VANTAGEM DO DRT: Resolve processos com τ similares que aparecem
  como um semicírculo achatado no Nyquist.

3.2 Circuito equivalente ↔ Picos DRT:

  ┌──────────────────────┬───────────────────────────────────┐
  │ Circuito selecionado │ DRT esperado                      │
  ├──────────────────────┼───────────────────────────────────┤
  │ Randles-CPE-W        │ 1 pico + cauda difusiva           │
  │ Two-Arc-CPE          │ 2 picos bem definidos             │
  │ Inductive-CPE        │ 1 pico + região negativa (rara)   │
  └──────────────────────┴───────────────────────────────────┘

  SE INCONSISTENTE:
  • Circuito com 1 arco, mas DRT com 2 picos:
    → O fitting de circuito escolheu modelo simplificado demais
    → Use BIC2 para ver se Two-Arc quase empatou
    → Na publicação, reporte ambos

  • DRT com 1 pico, mas circuito Two-Arc:
    → Overfitting do circuito
    → Verifique Bound_hits — parâmetros do 2º arco provavelmente
      convergiram para valores limites

3.3 R_inf (DRT) ↔ Rs_fit (EIS):

  EXPECTATIVA: R_inf ≈ Rs_fit
  POR QUÊ: Ambos estimam a resistência em alta frequência.

  DIFERENÇA GRANDE (>20%):
  • Faixa de frequência diferente entre os métodos
  • Fitting EIS pode incluir contribuição indutiva
  • DRT usa apenas parte imaginária, EIS usa real+imaginária

3.4 Soma γ_peak ↔ Rp_fit:

  EXPECTATIVA: Σ γ_peak ≈ Rp_fit
  POR QUÊ: Cada pico DRT corresponde à resistência de um processo.
  A soma deveria igualar a resistência total de transferência.

  DISCREPÂNCIA:
  • Picos muito largos → integração da área seria mais precisa
  • Regularização (λ) suprime amplitude → γ subestimado
  • Rp_fit inclui Warburg → Sigma contribui mas não é um pico

--------------------------------------------------------------------------------
4. CORRELAÇÕES DRT ↔ CICLAGEM
--------------------------------------------------------------------------------

4.1 Pico DRT principal ↔ Energia:

  • Pico em τ curto (< 1 ms) + γ alto → alta potência, energia moderada
  • Pico em τ longo (> 10 ms) + γ alto → processo lento contribui
    para energia mas limita potência

4.2 Múltiplos picos DRT ↔ Retenção:

  HIPÓTESE: Amostras com mais processos (mais picos) podem ser
  menos estáveis se um processo é instável.

  COMO VERIFICAR:
  • Tabela DRT+EIS: compare n_peaks com Retenção
  • Se n_peaks alto + Retenção baixa → um dos processos degrada
  • Identifique QUAL pico desaparece/muda → mecanismo de degradação

4.3 Largura dos picos ↔ Variabilidade energética:

  Picos DRT largos (width_decades > 1.5) indicam processos distribuídos.
  Estes tendem a causar maior flutuação ciclo-a-ciclo na energia.

--------------------------------------------------------------------------------
5. USANDO O HEATMAP DE CORRELAÇÃO
--------------------------------------------------------------------------------

O heatmap de correlação Spearman mostra TODAS as correlações de uma vez.

COMO LER:

  5.1 Identificar correlações FORTES (|ρ| > 0.7):
  • São as mais relevantes para publicação
  • Verifique o p-value (correlation_pvalues.csv): p < 0.05

  5.2 Identificar correlações INESPERADAS:
  • Rs ↔ Dispersion? → Materiais resistivos são mais heterogêneos?
  • n ↔ Retenção? → Expoente CPE prediz estabilidade?
  • Sigma ↔ Energia? → Difusão de Warburg limita energia?

  5.3 Cuidados:
  • Correlação ≠ causalidade
  • Com poucas amostras (<10), correlações podem ser artefatos
  • Outliers extremos podem dominar a correlação
  • Spearman é mais robusto que Pearson para relações não-lineares

CHECKLIST PARA PUBLICAÇÃO:
  □ Correlação ρ > 0.7 ou ρ < -0.7?
  □ p-value < 0.05?
  □ Sentido físico da correlação (explicação mecanística)?
  □ Consistente com a literatura?
  □ Verificado visualmente no scatter plot?

--------------------------------------------------------------------------------
6. COMO MONTAR UMA NARRATIVA CIENTÍFICA
--------------------------------------------------------------------------------

ESTRUTURA RECOMENDADA PARA UM ARTIGO/RELATÓRIO:

  1. CARACTERIZAÇÃO INICIAL (EIS):
     "As amostras foram classificadas em dois grupos pelo K-Means:
      Interface eficiente (Rs < X Ω, Rp > Y Ω) e Genérica estável.
      O PCA revela que PC1 (Z% da variância) é dominado por Rs e Q,
      enquanto PC2 (W%) separa por Rp e Sigma."

  2. CIRCUITO EQUIVALENTE:
     "O modelo Randles-CPE-W foi selecionado para Z% das amostras
      (BIC = ..., confiança = ...), consistente com interface
      eletrolítica com difusão semi-infinita. O expoente n = 0.85 ± 0.05
      indica dispersão frequencial moderada (superfície rugosa)."

  3. DESEMPENHO ENERGÉTICO (Ciclagem):
     "As amostras apresentaram energia específica de X-Y Wh/kg e
      potência de A-B W/kg, posicionando-se na região de supercapacitores
      no diagrama de Ragone. A retenção após 1000 ciclos foi de Z%."

  4. PROCESSOS FUNDAMENTAIS (DRT):
     "A análise DRT identificou dois processos de relaxação:
      τ₁ = 10⁻³ s (transferência de carga, γ₁ = X Ω) e
      τ₂ = 10⁻¹ s (difusão no filme, γ₂ = Y Ω)."

  5. CORRELAÇÕES:
     "A correlação de Spearman entre Rs_fit e potência
      (ρ = -0.82, p < 0.01) confirma que a resistência ôhmica
      é o fator limitante principal. A retenção correlaciona com
      Rank (ρ = 0.75), validando o score composto como preditor."

  6. CONCLUSÃO INTEGRADA:
     "A amostra [melhor Rank] combina a menor Rs (X Ω), maior Rp (Y Ω)
      e melhor retenção (Z%). O DRT revela que seu desempenho superior
      decorre de uma transferência de carga rápida (τ₁ mais curto) e
      menor contribuição difusiva (γ₂ menor), consistente com a
      microestrutura observada por [técnica complementar]."

--------------------------------------------------------------------------------
7. ERROS COMUNS DE INTERPRETAÇÃO
--------------------------------------------------------------------------------

  ❌ "Rp alto = ruim" → Depende do contexto!
     Para proteção contra corrosão, Rp alto é DESEJÁVEL.
     Para supercapacitores, Rp alto pode indicar pseudocapacitância.

  ❌ "n = 0.5 = Warburg" → Nem sempre!
     n = 0.5 no CPE é sugestivo, mas o circuito já tem Warburg explícito.
     O n do CPE modela a dispersão da interface, não a difusão.

  ❌ "Mais picos DRT = pior" → Não necessariamente!
     Mais processos podem significar mais mecanismos de armazenamento
     (ex: EDLC + pseudocapacitância = 2 picos = mais energia).

  ❌ "Rank 1 = melhor para tudo" → O rank é baseado em EIS apenas!
     Não considera custo, facilidade de síntese, toxicidade, etc.
     Use como GUIA, não como VERDADE ABSOLUTA.

  ❌ "BIC baixo = modelo correto" → BIC seleciona o MELHOR entre os
     candidatos. Se nenhum modelo é adequado, o BIC apenas escolhe
     o menos ruim. Verifique Res_estruturado e diagnósticos visuais.

--------------------------------------------------------------------------------
8. CHECKLIST FINAL PARA ANÁLISE COMPLETA
--------------------------------------------------------------------------------

  DADOS:
  □ Todos os arquivos foram importados corretamente?
  □ O scan rate está correto?
  □ Pelo menos 3 amostras para PCA/K-Means?

  EIS:
  □ Todos os fits convergiram (Sucesso = True)?
  □ Res_estruturado = False para a maioria?
  □ PCA tem variância explicada > 70% nos 2 primeiros PCs?
  □ Classificação K-Means faz sentido físico?

  CICLAGEM:
  □ Energia e Potência em faixas razoáveis para o material?
  □ Retenção calculada corretamente (amostras do mesmo grupo)?
  □ Ragone Plot consistente com literatura?

  DRT:
  □ Número de picos é consistente com o Nyquist?
  □ R_inf ≈ Rs_fit?
  □ Soma dos γ_peak ≈ Rp_fit?
  □ Picos persistem com diferentes λ?

  CORRELAÇÕES:
  □ Correlações significativas (p < 0.05)?
  □ Sentido físico verificado?
  □ Consistência entre EIS, Ciclagem e DRT?
  □ Resultados comparados com literatura?
