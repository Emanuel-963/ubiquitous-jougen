Capabilities

Everything a Battery Lab Needs

From raw impedance files to publication-ready reports in a few clicks.

📂

Multi-Vendor File Support

Read CSV, Gamry .dta, BioLogic .mpt/.mpr, Zahner .idf, and plain text without conversion.

🔌

Equivalent Circuit Fitting

11 circuits (Randles-CPE-W, ZARC-ZARC-W, Gerischer, Warburg Finite/Short…) with physics-based initial estimates. ML-assisted model selection.

📊

DRT Analysis

Tikhonov-regularised Distribution of Relaxation Times — model-free deconvolution of electrode processes.

✅

Kramers-Kronig Validation

Lin-KK residual test automatically flags non-stationary or non-linear spectra before fitting.

🔋

Battery Cycling Analysis

Ragone plots, capacity retention, energy/power vs. cycle, C-rate analysis from raw cycling data.

🔬

PCA & Ranking

Principal component analysis of fitted parameters. Automatic classification and sample ranking by health score.

📄

Branded PDF Reports

One-click PDF and Markdown reports with your logo, author name, and institution. AI-enhanced summaries optional.

⚡

Batch Processing

Process entire experiment folders in one run. Results persisted to a SQLite feature store for longitudinal tracking.

🐍

Python API

All functionality accessible programmatically. Integrate into notebooks, CI pipelines, or your own application.

Get Started

Up and Running in Minutes

Choose the installation method that fits your workflow.

Option 1 — Windows Installer (recommended for non-developers)

# Download and run:
IonFlow_Pipeline_Setup_0.3.1.exe

Option 2 — pip (Python 3.11+)

pip install ionflow-pipeline
# With BioLogic .mpr support:
pip install "ionflow-pipeline[biologic]"

Option 3 — Python API

from main import run_eis_pipeline

result = run_eis_pipeline() # reads data/raw/
print(result.feature_table.head()) # pandas DataFrame
